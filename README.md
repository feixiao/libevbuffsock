libevbuffsock is a minimal port of the evbuffer suite of functions from libevent to libev

###Ubuntu:
+ 下载[libev](http://dist.schmorp.de/libev/)

		编译过程
        ./configure
        make 
        sudo make install 
        
+ 下载[libevbuffsock](https://github.com/mreiferson/libevbuffsock)

		编译过程
        make 
        make install
